# Unity-Mobile-3D-Racing-Game
Gameplay Video Link : https://drive.google.com/file/d/1Yok4SbARUiE8vupj4K41nyH-jAO-JiRE/view?usp=drive_link

Assetlerin çoğu telifli olduğu için sadece script dosyalarını paylaştım.

------------------------------------------------------------ TR ------------------------------------------------------------
- Oyun verilerinin dosya tabanlı olarak okunması ve kaydedilmesi
- Gerçekçi ve arcade sürüş fiziği arasında geçiş yapılabilen kontrol sistemi
- Hızlanma avantajı sağlayan nitro, boost mekaniği
- Drift yapmaya dayalı sürüş mekaniği
- Farklı zorluk seviyelerine sahip yapay zekâ kontrollü rakipler
- Araç parçalarının geliştirilmesi ve değiştirilebilmesi
- Birincilik hedefli yarışlar ve süreye dayalı checkpoint tamamlama modları
- Farklı özelliklere sahip oynanabilir araç seçenekleri

 ------------------------------------------------------------ EN ------------------------------------------------------------
- File-based data loading and saving system
- Switchable driving physics between realistic and arcade modes
- Nitro, boost mechanic providing temporary speed advantages
- Drift based driving mechanics
- Ai controlled opponents with multiple difficulty levels
- Vehicle part upgrade and customization system
- Multiple race modes including first place finish and time based checkpoint challenges
- Multiple playable vehicles with distinct characteristics
